# Cursor 프롬프트: IA/메뉴구조도 기반 기능정의서 자동 생성

## 📋 요청사항

제공된 IA(Information Architecture)와 메뉴구조도를 기반으로, 첨부된 기능정의서 템플릿과 동일한 형식의 엑셀 문서를 자동으로 생성해주세요.

---

## 📄 입력 파일

1. **IA 문서** (또는 메뉴구조도): `[파일명]`
2. **기능정의서 템플릿**: `FunctionalSpecification_template.xlsx`

---

## 🎯 생성 목표

**IA/메뉴구조도의 각 메뉴/화면별로 상세 기능정의서를 자동 생성**

## 🔧 구현 요구사항

### 1. Python 스크립트 작성
```python
# openpyxl 사용
# IA 파일 파싱 → 메뉴 구조 추출
# 템플릿 기반 자동 생성
```

### 2. 자동 생성 로직

#### 메뉴/화면 추출
- IA 문서에서 메뉴 계층구조 파싱
- 각 메뉴항목을 화면으로 매핑
- 화면ID 자동 채번 (SCR001, SCR002, ...)

#### 화면별 시트 생성
- 템플릿 구조 복제
- 메뉴명 → 화면명 자동 입력
- 화면경로는 IA 계층 반영 (예: "관리자 > 사용자관리 > 목록")

#### 기본 컴포넌트 추론
화면 유형에 따라 기본 컴포넌트 자동 추가:
- **목록 화면**: 검색바, 필터, 데이터테이블, 페이징
- **상세 화면**: 정보 조회 영역, 수정버튼, 삭제버튼
- **등록 화면**: 입력 폼, 저장버튼, 취소버튼
- **대시보드**: 차트 영역, 통계 카드, 새로고침 버튼

#### 기본 기능 추론
컴포넌트 기반 기본 기능 자동 생성:
- 검색바 → 검색 실행 기능
- 등록버튼 → 등록 폼 열기/저장 기능
- 테이블 → 조회/정렬/페이징 기능

### 3. 서식 및 스타일 적용
- **헤더 행**: 굵게, 배경색 (RGB: 217,217,217)
- **섹션 제목**: 굵게, 크기 12pt, 배경색 (RGB: 189,215,238)
- **테두리**: 모든 데이터 셀에 가는 테두리
- **열 너비**: 내용에 맞게 자동 조정 (최소 10, 최대 50)
- **텍스트 정렬**: 
  - 헤더: 가운데 정렬
  - ID 컬럼: 가운데 정렬
  - 설명 컬럼: 왼쪽 정렬

### 4. 수식 및 자동화
- 화면ID: `="SCR" & TEXT(ROW()-1,"000")`
- 구성요소ID: `="COMP" & TEXT(ROW()-시작행,"000")`
- 기능ID: `="FUNC" & TEXT(ROW()-시작행,"000")`
- 목차 시트에 각 화면 시트로의 하이퍼링크 자동 생성

## ⚙️ 고급 옵션

### 화면 유형 자동 감지
메뉴명 키워드 기반 화면 유형 분류:
- "목록", "리스트" → 목록형 화면
- "등록", "생성", "추가" → 등록형 화면
- "상세", "조회" → 상세형 화면
- "대시보드", "현황" → 대시보드형 화면
- "설정", "관리" → 설정형 화면

